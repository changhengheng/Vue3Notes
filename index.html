# 介绍

  1. 是一门跨平台的编程语言

  2. 是JS的超集，除了包含JS，还有它自己的东西：泛型、接口等等

  3. TS不能直接在浏览器里使用，必须编译成JS代码才行

  4. 微软在2012年10月发布了公开版本，然后在2013年6月19日发布了正式版本

  5. 作者是安德斯·海尔斯伯格

## 特点

  1. 最终变编译成JS代码执行

  2. 强大的类型系统

  3. TS包含了JS的新特性
